== Description ==

Theme metabox options for Daisy Store theme.

== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be: './wp-content/plugins/daisy-store-companion/*'

Activate plugin at "Plugins" administration page.
